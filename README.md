# Occupation data

### Versions

Rails 6.0.3
Ruby 2.7.0

### Setup

```bash
git clone https://github.com/lukyans/occupation_data
cd occupation_data
```

Install all the gems: ```bundle install```

Create the database: ```rails db:create```

Migrate the database: ```rails db:migrate```
